Definition 2.3.10. Pangkat Matriks.

Misalkan \(A\) adalah matriks \(n\times n\text{,}\) dan misalkan \(r\in\Z\) adalah bilangan bulat. Kita mendefinisikan matriks pangkat \(A^r\) sebagai berikut:
\begin{equation*} A^r=\begin{cases} I\amp \text{jika } r=0;\\[2ex] \underset{r \text{ kali}}{\underbrace{AA\cdots A}}\amp \text{jika } r>0; \\[2ex] (A^{-1})^s \amp \text{jika } r=-s < 0 \text{ dan } A \text{ invertibel}. \end{cases}\text{.} \end{equation*}
